# Azure Cloud Engineering Portfolio

A lightweight, static portfolio website documenting my projects, blog posts, and progress in Azure Cloud Engineering.

## 🚀 Live Site
View the live portfolio here: [voyous.dev](https://voyous.dev/index.html)

## 🛠️ Tech Stack
* **HTML5**
* **CSS3**
* **GitHub Pages** (Hosting)

## 📂 Repository Structure
* `index.html` - Main landing page.
* `style.css` - Custom stylesheets.
* `projects.html` - HTML files for individual Azure project write-ups.
* `blog.html` - Main page for blog posts.
* `/posts` - Technical blog posts regarding Azure services and certifications.
* `/assets` - Images, diagrams, and PDF versions of my resume.

## 📝 Featured Content
* **Azure Projects:** Documentation on cloud architecture, networking, and security labs.
* **Technical Blog:** Deep dives into Azure concepts and my learning journey.
